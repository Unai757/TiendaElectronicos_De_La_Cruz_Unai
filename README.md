# Simulador de Tienda de Electrónica

Este es un ejercicio práctico de programación en Java que simula la gestión de una tienda de dispositivos electrónicos. El proyecto implementa  una lectura de archivo CSV, gestión de objetos y generación de informes en texto.

## Descripción

El programa carga automáticamente un inventario de productos desde un archivo CSV (stock.csv) al iniciar. A través de un menú interactivo en la consola, el usuario puede consultar productos, gestionar su carrito de compra y generar una factura final.

## Funcionalidades del Menú

1.  Ver Catálogo: Muestra la lista completa de productos disponibles (Móviles, Portátiles, etc.) con sus precios y stock actual.
2.  Comprar Producto: Solicita el ID del producto y la cantidad deseada. El sistema verifica el stock antes de añadirlo al carrito.
3.  Buscar Producto: Permite buscar un dispositivo específico por su ID para ver sus características técnicas detalladas.
4.  Finalizar Compra: Cierra la sesión de compra y genera un archivo de texto (facturas.txt) con el detalle de los productos adquiridos y el precio total.


### Instrucciones de uso:

1.  Abre el proyecto en el entorno de desarrollo  que utilizes (IntelliJ IDEA, Eclipse, etc.).
2.  Busca la clase Main.java.
3.  Ejecuta el programa.
4.  Sigue las instrucciones mostradas en la consola para navegar por el menú.
